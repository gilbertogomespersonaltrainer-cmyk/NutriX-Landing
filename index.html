Dúvidas

Perguntas frequentes

Preciso cancelar meu software atual para usar o NutriX? +
Não. O NutriX é focado em gestão de negócio: financeiro, agendamentos, WhatsApp e relacionamento com pacientes. Você pode continuar usando as ferramentas que já usa e adicionar o NutriX para gerir seu consultório.
Como funciona a integração com o WhatsApp? +
Você conecta seu número escaneando um QR Code diretamente nas configurações do NutriX. Em menos de 1 minuto seu WhatsApp está integrado e o sistema já começa a enviar lembretes automáticos.
Posso trocar de plano depois? +
Sim, você pode fazer upgrade ou downgrade a qualquer momento. O valor é ajustado proporcionalmente ao período restante.
Meus dados ficam seguros? +
Sim. O NutriX é multi-tenant com isolamento total de dados. Nenhuma nutricionista acessa os dados de outra. Todos os dados são criptografados e armazenados com segurança.
Tem contrato de fidelidade? +
Não. O NutriX é mês a mês sem fidelidade. Você pode cancelar quando quiser, sem multa e sem burocracia.
